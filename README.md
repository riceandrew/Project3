# Project3

Vegan Gems

Vegan Gems is an app designed to help users locate vegan-friendly restaurants as well as restaurants that offer the best vegan-friendly options.

In this app we utilized the following technologies: 
  1. React
  2. Next js
  3. Npm libraries: 
      a) google-maps-react
      b) express
      c) sequelize
  4. Bootswatch
  5. Node
  6. MySQL
  7. Google Maps API
